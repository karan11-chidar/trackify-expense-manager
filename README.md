# 💸 Expense Tracker System

A modern and responsive **Expense Tracker Web Application** built using **HTML, CSS, and JavaScript**.

This project helps users manage daily expenses, track spending, filter records, calculate statistics, and store data permanently using **LocalStorage**.
# 🚀 Features

## ✅ Expense Management

* Add expenses
* Edit expenses
* Delete expenses
* Dynamic expense cards

## 📊 Statistics Dashboard

* Total expenses
* Today's expenses
* Weekly expenses
* Monthly expenses

## 🔍 Smart Filters

* Search by description
* Filter by category
* Filter by date

## 💾 LocalStorage Support

* Data stays saved after page refresh
* No backend required

## 🎨 Modern UI

* Responsive design
* Hover animations
* Custom scrollbar
* Lucide icons integration
* Smooth user experience
